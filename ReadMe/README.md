# 책크
독서량 및 독서 인구 증가를 위한 독서 기록장 및 독서 SNS 모바일 앱입니다.  
프로젝트의 목표는 신규 독서 인구를 양성하기 위해 독서를 장려하고 독서에 대한 흥미를 유발하여 꾸준한 독서 활동으로 이어지는 것과, 독서 커뮤니티를 활용하여 기존 독서 인구 또한 공략하는 것입니다.

프로젝트 진행 기간 : 23.04 ~ 23.06
